## 프로젝트 소개
올바르고 안전한 타투문화를 전파하기 위해 시작된 프로젝트 입니다.

웹사이트를 통해 타투 도안과 타투이스트 정보를 제공합니다.

타투 시술에 사용된 정보는 안전하게 블록체인에 저장됩니다.

블록체인의 데이터는 커버업, 리터치, 부작용 원인파악 등에 이용됩니다.
